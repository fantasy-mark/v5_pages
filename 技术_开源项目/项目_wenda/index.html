    项目_wenda

    环境

    git clone https://github.com/wenda-LLM/wenda.git
cd wenda
# cuda环境(自行确保cuda driver >= cuda toolkit)
conda create --prefix=./venv python=3.10.9 cudnn=8.4.1.50 cudatoolkit=11.7.0 ffmpeg x264
conda activate ./venv
pip install torch==2.0.0+cu117 torchvision==0.15.1+cu117 torchaudio==2.0.1 --index-url https://download.pytorch.org/whl/cu117
pip install -r .\requirements\requirements.txt
pip install -r .\requirements\requirements-glm6b-lora.txt
# gcc 编译问题参照下w64devkit项目

    模型

    从huggingface下载项目所需模型，比如 text2vec-large-chinese \ chatglm-6b-int4 ，放置到新建的 model 目录

    配置项目

    重命名example.config.yml 为 config.yml配置cuda和数据精度，根据需要修改

    library:
   strategy: "calc:2 rtst:2 agents:0"
...
librarys:
...
   rtst:
...
      model_path: "model/m3e-base"
...
llm_type: glm6b
...
glm6b:
      path: "model/chatglm-6b-int4"
      #glm模型位置"
      strategy: "cuda fp16"
      #cuda fp16     所有glm模型 要直接跑在gpu上都可以使用这个参数
...
