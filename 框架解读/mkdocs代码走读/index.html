    • 问：为何cli.command可以代替click.command

    • 答：

    cli.command 并不能代替 click.command。在你之前的消息中，你提到了 mkdocs cli.command，我猜测你可能是想问 mkdocs 中的 cli.command 是什么意思。

    在 mkdocs 中，cli 是一个模块，它使用了 click 库来定义命令行接口。因此，当你看到 cli.command 时，它实际上是指 click.command。也就是说，在 mkdocs 的 cli 模块中，command 是从 click 库中导入的。

    • 答：

    是的，如果你使用 click.group 装饰器来定义 cli 函数，那么你就可以使用 cli.command 来定义子命令。

    click.group 装饰器会将 cli 函数转换为一个命令组，它具有 command 方法，可以用来定义子命令。例如：

    @click.group(context_settings={'help_option_names': ['-h', '--help']})
@click.version_option(
    __version__,
    '-V',
    '--version',
    message=f'%(prog)s, version %(version)s from { PKG_DIR } (Python { PYTHON_VERSION })',
)
@common_options
def cli():
    pass

@cli.command()
def subcommand():
    ...

    在这个例子中，我们使用 cli.command 装饰器来定义一个名为 subcommand 的子命令。

    # click.decorators.group
def group(
    name: t.Union[str, t.Callable[..., t.Any], None] = None, **attrs: t.Any
) -> t.Union[Group, t.Callable[[F], Group]]:
    """Creates a new :class:`Group` with a function as callback.  This
    works otherwise the same as :func:`command` just that the `cls`
    parameter is set to :class:`Group`.

    .. versionchanged:: 8.1
        This decorator can be applied without parentheses.
    """
    if attrs.get("cls") is None:
        attrs["cls"] = Group

    if callable(name):
        grp: t.Callable[[F], Group] = t.cast(Group, command(**attrs))
        return grp(name)

    return t.cast(Group, command(name, **attrs))

    # click.decorators.command
def command(
    name: t.Union[str, t.Callable[..., t.Any], None] = None,    # t.Union 联合体type
    cls: t.Optional[t.Type[Command]] = None,
    **attrs: t.Any,
) -> t.Union[Command, t.Callable[..., Command]]:
    r"""Creates a new :class:`Command` and uses the decorated function as
    callback.  This will also automatically attach all decorated
    :func:`option`\s and :func:`argument`\s as parameters to the command. """
